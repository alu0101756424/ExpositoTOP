\doxysubsection{Detailed Description}
Implementa el algoritmo de Bellman-\/\+Ford para cálculo de caminos mínimos. 

Dado una matriz de distancias y un número de nodos, calcula la distancia mínima desde el nodo 0 al nodo nodes-\/1, reconstruyendo además el camino. Se considera INFINITY para inicializaciones y se almacenan las aristas explícitamente para las relajaciones.
